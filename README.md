### General Info
***
 Full stack Money Manager app (SERN). Fetures: Add transactions, update transactions, delete transactions.

## Installation
***
# Database Script: 
In Backend Folder.
Name: budgetApp.sql

***
```
$ cd Backend npm install
$ cd Frontend npm install
```
# Run Backend 
```
$ cd Backend
$ npm test
```
# Run Frontend
```
$ cd Frontend
$ npm start
```
